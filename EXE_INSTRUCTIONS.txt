Auto Point Picker - Windows Application
=========================================

How to Run
----------
1. Ensure your input file is named "Final_Report.csv".
2. Place "Final_Report.csv" in the same folder as this application ("AutoPointPicker.exe").
3. Double-click "AutoPointPicker.exe".

What to Expect
--------------
- A black console window will appear briefly while processing.
- Two new files will be created in the same folder:
    1. "Auto Picked Top.csv"
    2. "Auto Picked Bottom.csv"
- The window will close automatically when finished.

Troubleshooting
---------------
If the window closes instantly and no files are created:
1. You may be missing the "Final_Report.csv" file.
2. The CSV file might be open in Excel (please close it and try again).
3. To see the specific error message:
    a. Open Command Prompt (cmd).
    b. Drag and drop the .exe file into the Command Prompt window.
    c. Press Enter.
    d. Read the error message displayed.
