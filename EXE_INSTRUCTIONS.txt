Top and Bottom Splitter - Windows Application
=============================================

How to Run
----------
1. Double-click "TopAndBottomSplitter.exe".
2. Click the "Browse..." button.
3. Select your input CSV file (usually "Final_Report.csv").
4. Click "Process File".

What to Expect
--------------
- The status log in the window will show the progress.
- Once complete, a success message will appear.
- Two new files will be created in the SAME FOLDER as your selected input file:
    1. "Auto Picked Top.csv"
    2. "Auto Picked Bottom.csv"

Troubleshooting
---------------
- If the application doesn't open, ensure you have permissions to run executables.
- If you see an error message, read the text in the "Log Output" area for details.
- Ensure the input file is not open in Excel while processing.
